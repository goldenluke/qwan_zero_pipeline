2. Teoria Formal QWAN

A hipótese fundamental postula a existência de um regime dinâmico intermediário, \(\mathcal{R}_{Q}\), no qual sistemas computacionais e biológicos maximizam sua capacidade adaptativa sob ruído.

2.1. Definição do Sistema

Considere um sistema estocástico discreto:

$$x_{t+1} = F(x_t; \xi) + \varepsilon_t$$

Onde \(x_t \in \mathbb{R}^n\) é o estado, \(\xi\) o parâmetro estrutural e \(\varepsilon_t \sim \mathcal{N}(0, \sigma^2)\).

2.2. Regimes Dinâmicos

Ordem Rígida:
$$\left| \frac{\partial F}{\partial x} \right| \ll 1$$ Estabilidade excessiva, baixa adaptabilidade.
Caos/Ruído:
$$\left| \frac{\partial F}{\partial x} \right| \gg 1$$ Alta sensibilidade, imprevisibilidade.
QWAN:
$$\left| \frac{\partial F}{\partial x} \right| \approx 1$$ Metaestabilidade computacional.

2.3. Formalismo Informacional

Propomos a função informacional estrutural \(\mathcal{I}(\xi)\):

$$\mathcal{I}(\xi) = H(x) \cdot \mathcal{F}(x)$$

Onde \(H(x)\) é a Entropia de Shannon (diversidade) e \(\mathcal{F}(x)\) a Informação de Fisher (ordem). O regime QWAN coincide com \(\xi^* = \arg\max_\xi \mathcal{I}(\xi)\).

Princípio da Metaestabilidade Computacional

"Sistemas adaptativos maximizam sua capacidade informacional na criticidade marginal (\(\lambda_{max} \approx 0\)), caracterizada pelo aumento da autocorrelação temporal e maximização conjunta de entropia e informação de Fisher."

A série hospitalar de SP opera tradicionalmente sob um \(\alpha \approx 0.07\), indicando anti-persistência extrema. Matematicamente, as barreiras do potencial estrutural (\(U(x)\)) são rígidas: as internações são controladas pela capacidade física instalada, forçando a reversão imediata à média.

Fenômenos Observados

  • Flickering: Oscilações de alta amplitude na variância móvel precedendo o salto de regime.
  • Metaestabilidade: O poço de potencial torna-se raso, facilitando a transição bi-estável para o estado de crise.
  • Borda do Caos: Ponto de transição onde a capacidade de processamento de novos pacientes degrada-se exponencialmente.

class UltraHealthEngine:
    def __init__(self, series):
        self.series = np.array(series, dtype=float)
        self.returns = np.diff(np.log(self.series + 1e-6))

    def dfa_alpha(self, ts):
        if len(ts) < 20:
            return 0.5
        y = np.cumsum(ts - np.mean(ts))
        n_vals = np.unique(np.logspace(1, 2, 8).astype(int))
        rms_vals = []
        for n in n_vals:
            segments = len(y) // n
            if segments == 0:
                continue
            rms = np.sqrt(np.mean([
                np.mean(
                    (y[i*n:(i+1)*n] -
                     np.polyval(np.polyfit(np.arange(n),
                                           y[i*n:(i+1)*n], 1),
                                np.arange(n)))**2
                )
                for i in range(segments)
            ]))
            rms_vals.append(rms)
        if len(rms_vals) < 2:
            return 0.5
        return np.polyfit(np.log(n_vals[:len(rms_vals)]),
                          np.log(rms_vals), 1)[0]

    def lempel_ziv(self, ts):
        if len(ts) < 5:
            return 0
        s = "".join(['1' if x > np.median(ts) else '0' for x in ts])
        n = len(s)
        i, u, v, w, c = 1, 1, 1, 1, 1
        while v + w <= n:
            if s[i:i+v] == s[i+w:i+v+w]:
                v += 1
            else:
                w += 1
                c += 1
                v = 1
        return (c * math.log2(n)) / n

    def fisher_info(self, ts):
        if len(ts) < 10:
            return 0
        h, _ = np.histogram(ts, bins=20, density=True)
        prob = h / (np.sum(h) + 1e-12)
        return 4 * np.sum(np.diff(np.sqrt(prob))**2)

    def permutation_entropy(self, ts, d=3):
        ts_arr = np.array(ts)
        if len(ts_arr) < d:
            return 1.0
        patterns = [tuple(np.argsort(ts_arr[i:i+d]))
                    for i in range(len(ts_arr)-d+1)]
        _, counts = np.unique(patterns, axis=0, return_counts=True)
        p = counts / counts.sum()
        return -np.sum(p * np.log2(p)) / np.log2(math.factorial(d))

    def multiscale_entropy(self, max_scale=10):
        ts = self.returns
        n = len(ts)
        entropies = []
        for scale in range(1, max_scale + 1):
            if n // scale < 5:
                entropies.append(1.0)
                continue
            rescaled = ts[:n - n % scale].reshape(-1, scale).mean(axis=1)
            entropies.append(self.permutation_entropy(rescaled))
        return np.array(entropies)

    def get_rolling_metrics(self, window=14):
        metrics = {
            'resilience': [], 'alpha': [], 'lz': [], 'fisher': [],
            'ac1': [], 'vol': [], 'skew': [], 'kurt': []
        }
        for i in range(window, len(self.returns)):
            slice_ts = self.returns[i-window:i]
            a = self.dfa_alpha(slice_ts)
            lz = self.lempel_ziv(slice_ts)
            f = self.fisher_info(slice_ts)
            ac1 = pd.Series(slice_ts).autocorr(lag=1)
            vol = np.std(slice_ts)
            ac1_norm = ((ac1 if not np.isnan(ac1) else 0) + 1) / 2
            risk = (ac1_norm + a +
                    (np.log10(vol**2+1e-9)+4)/4) / 3
            metrics['resilience'].append(1 - risk)
            metrics['alpha'].append(a)
            metrics['lz'].append(lz)
            metrics['fisher'].append(f)
            metrics['ac1'].append(ac1)
            metrics['vol'].append(vol)
            metrics['skew'].append(stats.skew(slice_ts))
            metrics['kurt'].append(stats.kurtosis(slice_ts))
        return {k: np.array(v) for k, v in metrics.items()}

@st.cache_data
def load_data(uf, year, month, cid, agg):
    try:
        sih = SIH()
        sih.load()
        files = sih.get_files(group='RD',
                              uf=uf,
                              year=year,
                              month=month)
        if not files:
            return pd.Series()
        df = sih.download(files[0]).to_dataframe()
        if cid:
            df = df[df['DIAG_PRINC']
                    .str.startswith(cid.upper(), na=False)]
        df['DT_INTER'] = pd.to_datetime(df['DT_INTER'],
                                        errors='coerce')
        ts = (df.groupby('DT_INTER')
                .size()
                .rename('hospitalizations')
                .sort_index())
        if agg == 'W':
            ts = ts.resample('W').sum()
        return ts
    except Exception as e:
        st.error(f"Data download error: {e}")
        return pd.Series()
