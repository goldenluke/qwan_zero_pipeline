Metaestabilidade como Princípio Computacional: Uma Teoria Formal do Regime QWAN

Fundamentação matemática para a maximização da capacidade adaptativa em sistemas de saúde coletiva através da criticidade marginal.

Artigo de Fundamentação Teórica | MARÇO, 2026 | FÍSICA ESTATÍSTICA & EPIDEMIOLOGIA COMPUTACIONAL

Abstract

Este tratado formaliza a hipótese de um regime dinâmico intermediário, denominado \(\mathcal{R}_{Q}\) (Regime QWAN), no qual sistemas computacionais maximizam sua capacidade adaptativa sob ruído. Afastando-se de slogans arquitetônicos, construímos um modelo matemático baseado em sistemas estocásticos discretos para definir a metaestabilidade computacional. Demonstramos que a eficiência informacional de uma rede de saúde é maximizada quando o maior expoente de Lyapunov se aproxima de zero, coincidindo com a maximização conjunta da Entropia de Shannon e da Informação de Fisher.

1. Definição do Problema e Sistema Base

A questão central reside na formalização de um regime dinâmico onde sistemas complexos (como a infraestrutura hospitalar) operam com máxima inteligência estrutural. Considere um sistema dinâmico discreto de base estocástica:

$$x_{t+1} = F(x_t; \xi) + \varepsilon_t$$

Onde \(x_t \in \mathbb{R}^n\), \(\xi\) representa o parâmetro estrutural e \(\varepsilon_t \sim \mathcal{N}(0, \sigma^2)\).

Classificação de Regimes Dinâmicos

I. Ordem Rígida: \(\left| \frac{\partial F}{\partial x} \right| \ll 1\). O sistema é altamente estável, porém possui baixa adaptabilidade. No contexto de saúde, representa a saturação administrativa total.
II. Caos e Ruído Dominante: \(\left| \frac{\partial F}{\partial x} \right| \gg 1\). Alta sensibilidade às condições iniciais e perda de previsibilidade. Representa o surto epidemiológico descontrolado.
III. Metaestabilidade (QWAN): \(\left| \frac{\partial F}{\partial x} \right| \approx 1\). A fronteira crítica entre estabilidade e instabilidade, onde a resposta adaptativa é otimizada.

2. Formalismo da Metaestabilidade Computacional

Um sistema computacional atinge o regime QWAN quando satisfaz simultaneamente condições de espectro de Lyapunov e dinâmica de correlação temporal.

2.1. Condição de Criticidade

$$\lambda_{max} \approx 0 \quad \text{e} \quad \frac{d}{dt} \text{Var}(x_t) > 0, \quad \rho(1) \to 1$$

Esta condição descreve o fenômeno de Lentidão Crítica (Critical Slowing Down): o sistema apresenta desaceleração na recuperação de choques sem atingir a divergência explosiva.

2.2. Função Informacional Estrutural

Definimos a métrica de inteligência estrutural \(\mathcal{I}(\xi)\) como o produto da diversidade estrutural pela sensibilidade paramétrica:

$$\mathcal{I}(\xi) = H(x) \cdot \mathcal{F}(x)$$

Onde \(H(x)\) é a Entropia de Shannon e \(\mathcal{F}(x)\) é a Informação de Fisher.

Princípio QWAN

"Sistemas adaptativos maximizam sua capacidade informacional quando operam em regime de criticidade marginal, caracterizado por expoente de Lyapunov nulo e maximização conjunta de entropia e ordem estatística."

class UltraHealthEngine:
    def __init__(self, series):
        self.series = np.array(series, dtype=float)
        # Log-returns: Captura a taxa de aceleração do sistema (momentum)
        self.returns = np.diff(np.log(self.series + 1e-6))

    def dfa_alpha(self, ts):
        """Detrended Fluctuation Analysis (DFA): Mede a memória fractal."""
        if len(ts) < 20: return 0.5
        y = np.cumsum(ts - np.mean(ts))
        n_vals = np.unique(np.logspace(1, 2, 8).astype(int))
        rms_vals = []
        for n in n_vals:
            segments = len(y) // n
            if segments == 0: continue
            # Detrending linear em cada segmento
            rms = np.sqrt(np.mean([np.mean((y[i*n:(i+1)*n] - np.polyval(np.polyfit(np.arange(n), y[i*n:(i+1)*n], 1), np.arange(n)))**2) for i in range(segments)]))
            rms_vals.append(rms)
        if len(rms_vals) < 2: return 0.5
        return np.polyfit(np.log(n_vals[:len(rms_vals)]), np.log(rms_vals), 1)[0]

    def lempel_ziv(self, ts):
        """Mede a inovação algorítmica (Complexidade LZ)."""
        if len(ts) < 5: return 0
        s = "".join(['1' if x > np.median(ts) else '0' for x in ts])
        n = len(s)
        i, u, v, w, c = 1, 1, 1, 1, 1
        while v + w <= n:
            if s[i:i+v] == s[i+w:i+v+w]: v += 1
            else: w += 1; c += 1; v = 1
        return (c * math.log2(n)) / n

    def fisher_info(self, ts):
        """Informação de Fisher: Mede o nível de ordem e nitidez estatística."""
        if len(ts) < 10: return 0
        h, _ = np.histogram(ts, bins=20, density=True)
        prob = h / (np.sum(h) + 1e-12)
        return 4 * np.sum(np.diff(np.sqrt(prob))**2)

    def permutation_entropy(self, ts, d=3):
        """Entropia de Permutação: Previsibilidade baseada na ordem dos dias."""
        ts_arr = np.array(ts)
        if len(ts_arr) < d: return 1.0
        patterns = [tuple(np.argsort(ts_arr[i:i+d])) for i in range(len(ts_arr)-d+1)]
        _, counts = np.unique(patterns, axis=0, return_counts=True)
        p = counts / counts.sum()
        return -np.sum(p * np.log2(p)) / np.log2(math.factorial(d))

    def reconstruct_potential(self):
        """U(x): Paisagem de estabilidade baseada na distribuição de Boltzmann."""
        hist, bins = np.histogram(self.returns, bins=30, density=True)
        centers = (bins[:-1] + bins[1:]) / 2
        sigma = np.std(self.returns)
        p = hist + 1e-12
        U = - (sigma**2 / 2) * np.log(p)
        return centers, U

    def multiscale_entropy(self, max_scale=10):
        """Analisa a entropia em diferentes escalas temporais."""
        ts = self.returns
        n = len(ts)
        entropies = []
        for scale in range(1, max_scale + 1):
            if n // scale < 5:
                entropies.append(1.0)
                continue
            rescaled = ts[:n - n % scale].reshape(-1, scale).mean(axis=1)
            entropies.append(self.permutation_entropy(rescaled))
        return np.array(entropies)

    def get_rolling_metrics(self, window=14):
        """Calcula a evolução temporal de todos os sensores de complexidade."""
        metrics = {
            'resilience': [], 'alpha': [], 'lz': [], 'fisher': [],
            'ac1': [], 'vol': [], 'skew': [], 'kurt': []
        }
        for i in range(window, len(self.returns)):
            slice_ts = self.returns[i-window:i]
            a = self.dfa_alpha(slice_ts)
            lz = self.lempel_ziv(slice_ts)
            f = self.fisher_info(slice_ts)
            ac1 = pd.Series(slice_ts).autocorr(lag=1)
            vol = np.std(slice_ts)

            # Resilience Index: Combinação de memória, lentidão e volatilidade
            ac1_norm = ((ac1 if not np.isnan(ac1) else 0) + 1) / 2
            risk = (ac1_norm + a + (np.log10(vol**2+1e-9)+4)/4) / 3

            metrics['resilience'].append(1 - risk)
            metrics['alpha'].append(a)
            metrics['lz'].append(lz)
            metrics['fisher'].append(f)
            metrics['ac1'].append(ac1)
            metrics['vol'].append(vol)
            metrics['skew'].append(stats.skew(slice_ts))
            metrics['kurt'].append(stats.kurtosis(slice_ts))

        return {k: np.array(v) for k, v in metrics.items()}

@st.cache_data
def load_data(uf, ano, mes, cid, agg):
    try:
        sih = SIH(); sih.load()
        files = sih.get_files(group='RD', uf=uf, year=ano, month=mes)
        if not files: return pd.Series()
        df = sih.download(files[0]).to_dataframe()
        if cid:
            df = df[df['DIAG_PRINC'].str.startswith(cid.upper(), na=False)]
        df['DT_INTER'] = pd.to_datetime(df['DT_INTER'], errors='coerce')
        ts = df.groupby('DT_INTER').size().rename('internacoes').sort_index()
        if agg == 'W': ts = ts.resample('W').sum()
        return ts
    except Exception as e:
        st.error(f"Erro ao descarregar dados: {e}")
        return pd.Series()
