Fundamentação Matemática

Abaixo detalhamos o motor analítico do LabSUS. O sistema integra aproximadamente 40 módulos especializados, dos quais os pilares algorítmicos estão formalizados nesta seção.

1. Normalização Dinâmica (ETL)

Aplica Interpolação por Spline Cúbica para garantir denominadores populacionais contínuos:

$$S_i(x) = a_i(x-x_i)^3 + b_i(x-x_i)^2 + c_i(x-x_i) + d_i$$

A taxa final (\(I\)) para qualquer indicador municipal é normalizada pela estimativa \(\hat{P}\) do instante \(t\): $$I_t = \frac{\sum \text{Eventos}_t}{\hat{P}_t} \times 10^k$$

2. Geometria de Agrupamento

Otimização via algoritmo de Lloyd para minimização da inércia intra-cluster:

$$\min_{S} \sum_{i=1}^{k} \sum_{x \in S_i} ||x - \mu_i||^2$$

A Lógica de Arquétipos rotula grupos medindo a Distância Euclidiana para vetores ideais \(A\): $$Label(C_i) = \text{arg min}_{j} \sqrt{\sum_{d=1}^D (C_{id} - A_{jd})^2}$$

3. Decomposição de Séries (Prophet)

Modelo aditivo generalizado para predição de demanda ou arrecadação:

$$y(t) = g(t) + s(t) + h(t) + \epsilon_t$$

Sazonalidade (\(s(t)\)) modelada via Séries de Fourier truncadas: $$s(t) = \sum_{n=1}^N (a_n \cos(\frac{2\pi nt}{P}) + b_n \sin(\frac{2\pi nt}{P}))$$

4. Probabilidade de Desfecho (Logit)

Mapeamento linear para o espaço sigmoide \((0,1)\):

$$P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \sum \beta_i x_i)}}$$

A explicabilidade é calculada via Odds Ratio (\(e^{\beta}\)) e valores SHAP baseados na Teoria dos Jogos: $$\phi_i = \sum_{S \subseteq \{x_1, \dots, x_n\} \setminus \{x_i\}} \frac{|S|!(n-|S|-1)!}{n!} [f(S \cup \{x_i\}) - f(S)]$$

5. Teoria de Grafos (Fluxos)

Centralidade de Intermediação para identificar hubs estruturais de saúde ou crédito:

$$g(v) = \sum_{s \neq v \neq t} \frac{\sigma_{st}(v)}{\sigma_{st}}$$

Onde \(\sigma_{st}\) é o número total de caminhos mínimos e \(\sigma_{st}(v)\) os que passam por \(v\).

6. Auditoria de Anomalias

Identificação de outliers em faturamento ou notificações via Z-Score robusto:

$$Z = \frac{x - \text{median}(x)}{\text{MAD}(x) \times 1.4826}$$

Utiliza o desvio absoluto da mediana (MAD) para evitar que erros de preenchimento contaminem o limiar de detecção.

LabSUS na Gestão Fazendária

O Padrão de Projeto Estatal permite transpor os 40 módulos para o domínio fiscal. Substituímos variáveis epidemiológicas por métricas tributárias, mantendo a integridade matemática da arquitetura.

Exemplo: Yield Tributário Municipal
$$\text{Tax\_Yield}_{mun} = \frac{\text{Arrecadação}_{real}}{\widehat{\text{PIB}}_{spline}}$$
