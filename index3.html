Fundamentação Matemática

Detalhes dos pilares algorítmicos que regem os ~40 modelos especializados do motor analítico src/.

1. Normalização Dinâmica (ETL)

Garante denominadores populacionais contínuos via Spline Cúbica:

$$S_i(x) = a_i(x-x_i)^3 + b_i(x-x_i)^2 + c_i(x-x_i) + d_i$$

Taxa final normalizada: $$I_t = \frac{\sum \text{Eventos}_t}{\hat{P}_t} \times 10^k$$

2. Geometria de Agrupamento

Minimização da inércia intra-cluster para perfis estatais:

$$\min_{S} \sum_{i=1}^{k} \sum_{x \in S_i} ||x - \mu_i||^2$$

Distância Euclidiana para Arquétipos: $$Label(C_i) = \text{arg min}_{j} \sqrt{\sum (C_{id} - A_{jd})^2}$$

3. Decomposição de Séries (Prophet)

Modelo aditivo generalizado para predição de demanda:

$$y(t) = g(t) + s(t) + h(t) + \epsilon_t$$

Sazonalidade via Fourier: $$s(t) = \sum_{n=1}^N (a_n \cos(\frac{2\pi nt}{P}) + b_n \sin(\frac{2\pi nt}{P}))$$

4. Probabilidade e Explicabilidade

Mapeamento linear para o espaço sigmoide e valores SHAP:

$$P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \sum \beta_i x_i)}}$$

Fatores de risco via Odds Ratio: \(e^{\beta}\).

LabSUS na Gestão Fazendária

A transposição do motor analítico permite processar fluxos fiscais com o mesmo rigor. Substituímos variáveis epidemiológicas por métricas tributárias, mantendo a integridade matemática da arquitetura full-stack.

Exemplo: Yield Tributário Municipal
$$\text{Tax\_Yield}_{mun} = \frac{\text{Arrecadação}_{real}}{\widehat{\text{Base\_Tributável}}_{spline}}$$

Equivalência Funcional (SUS ↔ Fazenda)

Survival analysis → Tempo até Inadimplência
Epidemic Surges → Surtos de Evasão Fiscal
Patient Journey → Ciclo de Vida do Contribuinte
